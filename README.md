# Syed Hammad Ali — Creative Portfolio

Public portfolio website for Syed Hammad Ali, Creative Manager and multidisciplinary designer.

**Live website:** https://syedhammada321-lang.github.io/syed-hammad-portfolio/

## Stack

- Next.js 15 with static export
- React and TypeScript
- Tailwind CSS v4
- Framer Motion and GSAP
- Lucide React
- Self-hosted Space Grotesk and Manrope

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The exported website is written to `out/` and is deployed automatically through GitHub Actions.

## Updating portfolio content

- Project content: `lib/projects.ts`
- Homepage content: `components/portfolio-home.tsx`
- Images: `public/assets/`
- Design system: `app/globals.css`

The current project imagery and case-study copy are labelled concept studies and can be replaced with approved client work later.
